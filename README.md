# img2map

A program for converting an image part into HTML map poly points

## How to run

- Download the latest release
- Unzip it
- Run img2map or img2map.exe depending on your system

## How to build

1. You'll need a C/C++ compiler and CMake
2. Run 
```bash
git clone https://github.com/SemkiShow/img2map
cd img2map
git submodule update --init --recursive --depth 1
chmod +x run.sh
./run.sh
```
